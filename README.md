# Documents Service

This service will handle all operations related to contracts:
 - Template management
 - Document management
 - Document signing

### Setup api -> ruby version 2.6.6

1.- Git clone repo.

2.- Execute

```sh
bundle install
```

3.- Copy example.env to .env

```sh
cp example.env .env
```

4.- Run server

```sh
bundle exec rackup
```

6.- Run test cases

```sh
bundle exec rake test
```
